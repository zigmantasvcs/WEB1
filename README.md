## Savarankiðkos uþduotys

* Paprastos: paskaitos metu kartu su lektoriumi atliktø uþduoèiø patobulinimai.
* Namø darbai: Ðiø uþduoèiø turësite daugiausiai, nuo paprasèiausiø skirtø elementarioms ágytoms þinioms uþtvirtinti iki sudëtingø kompleksiniø uþduoèiø.
* Projektas: I projekto etapas yra iðsigryninti idëjà, schematiðkai nusibraiþyti idëjà popieriaus lape, sukurti idëjos dizainà - veikiantèià statinæ svetainæ. Ðis etapas skirtas kaip pasiruoðimas antràjam etapui.
